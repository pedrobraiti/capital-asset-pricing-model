# capital-asset-pricing-model

Empirical replication and out-of-sample extension of **Fama & French (2004), "The Capital Asset
Pricing Model: Theory and Evidence"** (*Journal of Economic Perspectives*, 18(3):25–46), using the
Kenneth R. French Data Library.

> Work in progress — results, charts and full write-up are generated by the pipeline and embedded here.

## How to run

```powershell
py -3.12 -m venv .venv
& ".venv\Scripts\Activate.ps1"
pip install -r requirements.txt
pytest -q
python scripts/run_study.py
python scripts/build_figures.py
```

## Stack

Python 3.12 · pandas · numpy · scipy · matplotlib. Data: Kenneth R. French Data Library.
